# 💳 Credit Card Fraud Detection

A machine learning web app that predicts whether a credit card transaction is fraudulent.

## 🚀 Features
- Fraud prediction using a trained machine learning model
- Interactive Streamlit web app
- Input transaction features manually
- Real-time prediction result
- Clean deployed interface

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit
- Joblib

## 💡 How It Works
1. User enters transaction feature values
2. Model analyzes the values
3. App predicts whether the transaction is fraudulent
4. Result is displayed instantly
